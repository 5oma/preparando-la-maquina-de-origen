
Es importante para la invocabilidad de este sistema que exista cierta sistematización en los nombres.

Para invocar necesitamos nombrar, pero para nombrar no podemos simplemente dejar que se acumulen los nombres sino proponer cierto sistema.

El principal costo puede venir de la rigidez.   

Aquí podemos ir decidiéndonos por sistemas para nombrar. 

Cada mundo de nombres o sistemas de nombres nos lleva a diferentes mundos o dominios del sistema. 

Conforme agreguemos items, estos deberán ser cada vez nombrados de mejor manera para garantizar la invocabilidad.
